Setup Guide · One Sheet

Job Application Toolkit
Your own free instance in 15 minutes

Built by Cara A. Brown  ·  www.getclarity.consulting
This toolkit runs entirely in your browser — no servers, no subscriptions. You host your own copy on GitHub for free. The only cost is the Anthropic API, which bills by usage (typically pennies per session). Here's how to get set up.
01

Create a GitHub account

Go to github.com and sign up for a free account if you don't have one. Your username will be part of your toolkit's URL, so pick something professional.

02

Fork the toolkit repository

Go to github.com/caraabrown/jobsearchtoolkit and click the Fork button in the top right. This creates your own personal copy of the toolkit under your GitHub account. You are not editing the original — it's entirely yours.

03

Enable GitHub Pages

In your forked repo, go to Settings → Pages. Under "Source," select Deploy from a branch, choose main and / (root), then click Save. Wait about 60 seconds.

Your live URL will be: yourusername.github.io/jobsearchtoolkit
04

Get your Anthropic API key

The AI features require an Anthropic API key. Go to console.anthropic.com, create a free account, then navigate to API Keys → Create Key. Copy the key — you only see it once. Add a small credit ($5–10) under Billing to get started. Typical usage is a few cents per session.

05

Add your API key to the toolkit

Open your live toolkit URL. Click ⚙ API Key in the top right corner, paste your key, and click Save. The key is stored only in your browser — it never leaves your device except to call the AI directly.

06

Upload your resumes

Go to Step 00 — My Resumes in the left nav. You'll see slots for each resume version. Click ⬆ Upload PDF / TXT to upload your resume file, or use ✎ Paste Text to paste the content directly. The toolkit will use your versions for every analysis. Re-upload any time you make updates.

  • Best format: PDF exported from Google Docs or Word
  • Tip: Upload a separate version for each role type you're targeting
  • Privacy: Resumes are saved in your browser only — not on any server
07

Start your job search

Go to Step 01 — Job Description, paste a job posting, and work through the steps in order. The toolkit handles ATS scoring, line-by-line resume edits, cover letters, interview prep, company research, salary intelligence, and application tracking — all in one place.

Note: Your data (resumes, job tracker, settings) lives in your browser's local storage. Clearing your browser data will erase it. Export or screenshot anything important.
